A complete web-based appointment management system for beauty salons. Streamline your salon operations from booking to payment processing.

## ✨ Features

### 👤 Customer Features
- 📅 **Book Appointments** - Easy online booking for various salon services
- 💳 **Payment Integration** - Secure PayFast payment gateway (Sandbox mode)
- ✅ **Booking Confirmation** - Instant confirmation with payment option
- 🎨 **Service Gallery** - Browse available hairstyles and treatments

### 👑 Admin Features
- 📊 **Admin Dashboard** - Complete overview of all appointments
- 📋 **Queue Management** - Live queue status with real-time updates
- 📈 **Analytics Dashboard** - Visual insights with Chart.js (bar, pie, line charts)
- 🔄 **Status Management** - Update appointment status (Booked → In Progress → Completed)
- 🗑️ **Appointment Management** - Delete or modify appointments

### 🔐 Security Features
- 👤 **User Authentication** - JAAS-based login system (GlassFish File Realm)
- 🔑 **Role-Based Access** - USER and ADMIN roles with different permissions
- 🚫 **Access Control** - Users can't access admin pages, admins can't access booking pages
- 📝 **User Registration** - New users can create accounts

## 🛠️ Technology Stack

| Technology | Purpose |
|------------|---------|
| **Java 8+** | Backend logic |
| **JSP** | Dynamic web pages |
| **Servlets** | Request handling |
| **GlassFish 7** | Application server |
| **Derby Database** | Data storage |
| **Chart.js** | Data visualization |
| **PayFast** | Payment gateway |
| **HTML5/CSS3** | Frontend styling |
